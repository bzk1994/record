# gulpConfig
**gulpConfig配置**
*（为多页面应用配置）*

##应用npm包

gulp-htmlmin			html压缩

gulp-minify-css		css压缩

gulp-uglify			js混淆压缩

gulp-imagemin			image压缩

gulp-rev-append		页面的引用添加版本号，清除页面引用缓存

gulp-sourcemaps		css map

gulp-notify			当发生异常时提示错误

gulp-plumber			当发生异常时提示错误

imagemin-pngquant	深度压缩png图片

gulp-autoprefixer	添加css前缀

gulp-zip				压缩zip

gulp-cache			图片缓存***
